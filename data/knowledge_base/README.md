# Knowledge Base

Place `.txt`, `.md`, and `.pdf` veterinary documents in this directory.

Optional metadata frontmatter for `.txt` and `.md` files:

```text
---
breed: Labrador Retriever
species: dog
source: Merck Manual
title: Labrador Mobility Notes
topic: orthopedic
---
```

If no frontmatter is present, the ingestion pipeline defaults documents to `universal`.
